<!-- # Weather-Dashboard

This is a weather dashboard that allows the user to look up any city's current weather and forecast. Use the search bar on the top left to look up any city and click search. Immediately that city's current temperature, high, low, humidity, wind speed and UV index will be displayed. 
Below the current weather is a 5 day forecast displaying the date, temperature and humidity. 

The user's search history is stored locally and displayed below the search bar. A user can easily click a prior searched city and that city's updated weather information will be prominately displayed. 

RESOURCES

This page was built using the Open Weather API. The API was used to generate all the information displayed in the current weather and 5-day forecast.
